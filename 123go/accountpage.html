1
顶部导航栏Header Navigation
HEADER
头像显示用户头像；未上传头像时显示平台吉祥物图示。
余额显示格式：「币别代码 + 金额」，例：PHP ₱808,957；点击旋转图示可刷新余额。
存款按钮显示「+ Deposit」按钮，点击跳转至「存款」页面。
客服图示点击后开启客服入口（在线客服 / ChatBot）。
信件图示点击后跳转至「站内信 / 通知中心」页面；有未读时显示红点图标。
2
VIP 专属权益VIP Exclusive Privileges
VIP BAR
VIP 系统 / 用户等级配置
显示内容显示「VIP Exclusive Privileges」文案 + 右侧箭头图示。
点击行为点击后跳转至「VIP 专属权益」详情页面。
显示条件后台启用 VIP 功能时显示；停用时整条隐藏，不留白。
3
余额 / 存提款Balance & Deposit / Withdraw
BALANCE
3.1 余额显示
标题文案固定显示「Balance」。
余额格式硬币图示 + 千分位金额，保留两位小数，例：🪙 649,734,464.00。
币别依当前用户币别显示对应余额。
刷新页面进入时自动刷新；下拉页面可手动刷新。
3.2 Deposit / Withdraw 按钮
Deposit 按钮图示 + 「Deposit」文案，点击跳转至「存款」页面。
Withdraw 按钮图示 + 「Withdraw」文案，点击跳转至「提款」页面。
排列方式两个按钮横向等宽并排，中间有分隔线。
显示条件存款 / 提款功能依后台配置决定是否显示；功能停用时对应按钮隐藏。
4
推广 BannerPromo Banner
PROMO BANNER
活动配置 / 个人中心 Banner 管理
数据来源后台配置的个人中心专属 Banner 图片及跳转链接。
图片比例依设计稿固定比例显示；加载失败时显示预设占位图。
点击行为点击后跳转至后台配置的目标页面（站内或外部链接，外部以新视窗开启）。
无数据后台未配置或停用时，Banner 整区隐藏，不留白。
5
Services服务项目
SERVICES
5.1 区块说明
区块标题固定显示「Services」文案。
排列方式3 列网格，每格显示图示 + 文案，依序排列。
图示加载图示由前端内建；加载失败时显示预设占位图示。
5.2 各项目说明
项目点击行为 / 说明
Game (History)跳转至「游戏历史记录」页面，显示用户的历史游戏投注记录。
Transaction跳转至「交易记录」页面，显示存款、提款、转账等流水记录。
Rewards跳转至「奖励中心」页面，显示可领取及已领取的奖励项目。
Recent Played跳转至「最近游玩」页面，显示用户近期游玩的游戏列表。
Favorites跳转至「我的收藏」页面，显示用户收藏的游戏列表。
Points Mall跳转至「积分商城」页面，可使用积分兑换奖品或优惠。
Task Center跳转至「任务中心」页面；图示右上角显示待完成任务数量角标（红点）。
Invite Friends跳转至「邀请好友」页面；图示附有 HOT 标签。
Become Affiliate跳转至「成为代理」页面或开启代理申请流程。
My Voucher跳转至「我的优惠券」页面，显示用户持有的优惠券列表。
Redeem跳转至「兑换码」页面，可输入兑换码领取奖励。
Result跳转至「开奖结果」页面，显示各游戏历史开奖记录。
Sports Rules跳转至「体育规则」页面，显示体育投注相关规则说明。
HOT 标签依后台配置控制，后台标记为 HOT 的项目图示右上角显示红色 HOT 角标。
任务中心角标依用户未完成任务数量实时显示，无未完成任务时角标隐藏。
6
Settings设置项目
SETTINGS
6.1 区块说明
区块标题固定显示「Settings」文案。
排列方式3 列网格，每格显示图示 + 文案。
6.2 各项目说明
项目点击行为 / 说明
Account Security跳转至「帐户安全」页面，可修改密码、绑定手机 / 邮箱、双重验证等安全设置。
Self-Limitation跳转至「自我限制」页面，可设定存款上限、投注上限、登入限制等自我管理选项。
Preference点击后由下往上展开「Preference」设置弹窗（详见第 6.3 章节）。
6.3 Preference 弹窗规格
PREFERENCE POPUP
点击「Preference」后，弹窗由下往上滑入（半屏弹层）；点击弹窗外部区域或右上角「×」关闭弹窗。
Skin Settings切换日 / 夜间模式（Light / Dark）；右侧 Toggle 开关，当前状态实时套用至全局主题。
Language 单选语系，选项：简体中文、English、Tiếng Việt、Filipino、Cebuano。
切换后页面文案即时更新为对应语系；预设依系统语言自动选中。
Odds Setting 单选赔率显示格式：EU Odds（欧洲盘） 或 HK Odds（香港盘）。
切换后体育相关页面的赔率显示即时更新。
Page Layout Switch 切换游戏列表的版面布局，三种模式：
  • 新手模式：3 列大图格，适合初学者浏览。
  • 亚洲模式：混合大小图格，信息密度中等。
  • 欧洲模式：列表横向宽图格，信息密度最高。
切换后娱乐城等游戏列表页面立即套用对应版面。
关闭方式点击右上角「×」或点击弹窗外部区域，收起弹窗并保存已选设置。
设置保存用户调整后自动保存至用户配置，下次进入时保持上次设置。

Others其他
OTHERS
7.1 区块说明
区块标题固定显示「Others」文案。
排列方式3 列网格，每格显示图示 + 文案。
7.2 各项目说明
项目说明 / 点击行为
Policies图示为平台 Logo；点击后跳转至「平台政策」页面（含条款、隐私权、负责任博彩等）。
Ver 1.2.11 显示当前 APP 版本号，格式：「Ver X.X.XX」，版本号由前端内建（或从 API 取得）。

点击后跳转至版本更新说明页面或触发更新检查流程。
App Download点击后跳转至「下载 APP」页面或直接触发 APP 下载 / 安装流程。
8
登出Log Out
LOG OUT
显示文案固定显示「Log Out」。
点击行为点击后弹出确认提示框（Confirm Dialog），询问用户是否确认登出。
确认登出清除登入 Token 及本地 Session，跳转至「首页（综合）」页面。
取消关闭确认提示框，留在个人中心页面，不做任何操作。
按钮样式独立一行，文字居中，依设计稿配色（蓝色文字）。
登出操作需有确认弹窗保护，不可直接单击即登出，防止误触。

SportsSports 图示 + 文案点击跳转至「体育大厅」页面。
